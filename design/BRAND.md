# Vibrant Pulse - Design System

## Brand Identity
**Vibrant Pulse** is a modern fintech design system that balances high-energy, bright colors with a structured, professional layout. It uses a harmonious palette of blues, violets, and teals to create an interface that feels both exciting and trustworthy.

## Colors

### Primary Palette
- **Surface:** `#faf8ff` (Light, airy base)
- **Surface-Bright:** `#faf8ff`
- **Surface-Dim:** `#d2d9f4`
- **Surface-Container:** Used for cards and secondary backgrounds to create soft hierarchy.

### Accents
- **Primary:** `#0062ff` (Vibrant blue for main actions)
- **Secondary:** Shades of violet and teal designed to match the primary blue's saturation and brightness.

## Typography
- **Primary Font:** **Montserrat**
- **Characteristics:** Geometric sans-serif, highly legible for financial data, friendly yet professional.

## UI Components Style
- **Roundness:** `ROUND_EIGHT` (8px corner radius) for a modern, approachable feel.
- **Visual Rhythm:** Generous whitespace and soft shadows to ensure data-heavy screens remain scannable and clean.
- **Interactivity:** High-contrast primary buttons and subtle hover states using the vibrant accent palette.
